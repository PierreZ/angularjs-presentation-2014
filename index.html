Pourquoi développer un nouveau framework ? 2/2

Il existe de nombreuses (trop ?) de framework/bibliothèques pour faire du web dynamique côté client

Mais AngularJS c'est quoi ?

  • Framework JavaScript développé par Google depuis 2010
  • Framework complet !
  • Conçu pour SPA (Single Page Application)
  • Propose un vrai modèle MVC (Modèle Vue Contrôleur)
  • Fonctionnalité de data binding en temps réel
  • Communauté très active
  • Open-source !
  • Compatible à partir de IE-8

Est-ce que c'est cool ?

Et je peux faire quoi avec ?

Vue d'ensemble d'AngularJS

Les composants de AngularJS

  • Directives -> Manipulation du DOM
  • Contrôleurs -> Lien entre le DOM et les données,création du scope
  • Services -> Contient le code métier, ainsi que la mémoire partagé de l'application
  • Bonus -> Animations, routes etc...
